# Real-Time Fluid Simulation in Unity

A particle-based fluid simulation built in Unity using C#.

## Inspiration
This project was initially inspired by Sebastian Lague's fluid simulation exploration:
https://www.youtube.com/watch?v=rSKMYc1CQHE

## Overview
The simulation models fluid behavior using interacting particles and real-time physics updates. The system tracks particle motion, velocity vectors, and collision behavior to approximate fluid dynamics.

## Current Features
- Real-time particle simulation
- Adjustable simulation parameters
- Data visualization of particle movement

## Future Improvements
- Hand-tracking interaction using camera input
- GPU acceleration for larger particle counts
- Implementation of Smoothed Particle Hydrodynamics (SPH)
- Interactive obstacles and environmental forces

## Technologies
- Unity
- C#
- Physics simulation
